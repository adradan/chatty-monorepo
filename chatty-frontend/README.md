# Chatty Frontend

Frontend built with Vite and React

## Getting Started

### Dependencies/Prerequisites

* [pnpm](https://pnpm.io/installation)

### Local Development Setup

```bash
pnpm install
pnpm dev

# To build for production
pnpm run build
```

This project uses Prettier for consistent formatting.
